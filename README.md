# Smart Parking System

A Smart Parking System built using Arduino.

## Components
- Arduino Uno
- IR Sensors
- LEDs
- Breadboard
- Jumper Wires

## Features
- Detects occupied parking spots
- Indicates free and occupied spaces
- Simple Arduino-based project
